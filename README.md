## Server-Side (Node.js) Authorization
```
tiddlywiki --listen readers="(anon)" writers="your_username" username="your_username" password="your_password"
```

! `(anon)`: A special token that grants access to everyone without a login.
! Authentication: When you want to edit, you navigate to the `/login-basic` route (e.g., `http://127.0.0.1:8080/login-basic`) to enter your credentials.
